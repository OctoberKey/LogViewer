# LogViewer
基于log4net UdpAppender 的实时日志查看工具

支持如下功能：

1. 支持同时收集多个APP的日志，并且提供筛选功能
2. 支持按照线程ID进行筛选日志
3. 支持通过日志名称筛选日志
4. 支持一键清空显示区域的所有日志
5. 支持自动滚动或者禁止自动滚动
6. 支持置顶或者取消置顶
7. 支持复制配置，复制配置之后粘贴到APP的配置文件中即可使用
8. 支持自定义监听端口
9. 支持调整窗口的透明度
10. 支持调整显示区域的字体大小
11. 增加翻页功能，目前每页最多显示1000条记录，超过之后可以通过翻页功能向前或向后查看
12. 支持搜索功能（搜索结果有一定的问题，数量会翻倍，再次点击问题消失）
13. 搜索结果支持高亮显示
